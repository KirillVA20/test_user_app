# Test User App

React-приложение для управления пользователями с системой аутентификации.

## Основные возможности

- Аутентификация пользователей (вход/выход)
- Просмотр списка пользователей
- Создание и редактирование пользователей
- Адаптивные карточки пользователей с аватарами

## Технологический стек

- **React 16** с TypeScript
- **React Router v6** для навигации
- **TanStack Query** для управления серверным состоянием
- **Ant Design** для UI-компонентов
- **styled-components** для стилизации
- **Axios** для HTTP-запросов
- **Webpack 5** для сборки

## Архитектура

Проект организован по методологии **Feature-Sliced Design**:

- `app/` — провайдеры и конфигурация приложения
- `pages/` — страницы (login, users, not-found)
- `features/` — бизнес-функции (auth, user management)
- `entities/` — бизнес-сущности (user)
- `shared/` — переиспользуемый код (ui, api, lib)

## Установка и запуск

- Создайте env файл, скопируя .env.example
- Выполните команды в терминале
```bash
npm install
npm start
```

## Особенности реализации

- Модульная структура с четким разделением ответственности
- TypeScript path aliases для удобного импорта (`@/`, `@entities/`, и т.д.)
- Защищенные роуты с редиректами
- Управление состоянием через React Context + TanStack Query
- Модальные окна для создания/редактирования пользователей
